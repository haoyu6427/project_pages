RA-YOLO: Resolution-Adaptive YOLO for Object Detection

Overview

Factors affecting object detection: spatial resolution, amplitude resolution, and object distance.

Abstract

Object detection is a key task in autonomous driving. Detection accuracy is influenced by image resolution, compression level, and object distance. We propose RA-YOLO—a resolution-adaptive YOLOv5 model—that dynamically adjusts feature scales based on input resolution to balance detection accuracy and computational cost. Our model was trained and evaluated on a mixed-resolution dataset derived from TJU, EuroCity, and BDD. RA-YOLO shows improved performance across varied conditions and is robust to moderate compression artifacts. It extends the reliable detection range and provides practical guidance for resolution-compression trade-offs in real-world deployments.

Method

The proposed RA-YOLO architecture dynamically adapts feature processing for different input resolutions.
